# Java Game

A simple Java game built with Swing.

## Building & Running

```bash
# Run the game
./gradlew lwjgl3:run

# Build a JAR
./gradlew lwjgl3:jar
```

The built JAR will be at `lwjgl3/build/libs/`.

## Project Structure

- `core/`: Shared game logic
- `lwjgl3/`: Desktop application

## Requirements

- Java 11+
- Gradle (included via wrapper)
